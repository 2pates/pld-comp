Detailed Description

The class for the control flow graph, also includes the symbol table.

A few important comments:

  • The entry block is the one with the same label as the AST function name. (it could be the first of bbs, or it could be defined by an attribute value)
  • The exit block is the one with both exit pointers equal to nullptr. (again it could be identified in a more explicit way)
